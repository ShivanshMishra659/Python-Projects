# Python Mini Projects

A small collection of beginner-friendly Python projects built while practicing core concepts — loops, conditionals, randomization, and input validation.

## Projects

### 1. Quiz Game
A command-line quiz where questions are presented in order. The user answers each one, earning +1 point per correct answer, with the final score shown at the end.

**Run:**
```bash
python quiz_game.py
```

### 2. Number Guessing Game
The user sets an upper limit, N. The program picks a random number between 0 and N, and the user guesses it. After each guess, the program tells you whether the number is higher or lower, and reports the total number of guesses once you find it.

**Run:**
```bash
python num_guess.py
```

## Requirements
- Python 3.x
- No external libraries needed

## How to Run
1. Clone this repository
   ```bash
   git clone <repo-url>
   ```
2. Navigate into the project folder
3. Run the script you want
   ```bash
   python <filename>.py
   ```
